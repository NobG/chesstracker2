# Story 01: Projektgrundgeruest

## Ziel

Ein neues, wartbares Spring-Boot-Projekt fuer chesstracker anlegen.

## Ausgangslage

Der alte chesstracker wird verworfen. Der Workspace startet leer.

## Akzeptanzkriterien

- Maven-Projekt mit Java 21 ist vorhanden.
- Spring-Boot-Struktur ist nach Controller, Service, Repository, Model, DTO und ViewModel getrennt.
- `application.yml` ist angelegt.
- README und Projektplanung existieren.

## Technische Hinweise

Kein alter Code wird uebernommen.

## Tests

Build und Smoke-Test spaeter mit verfuegbarem Maven ausfuehren.
